POLÍTICA DE PRIVACIDAD

Su privacidad es muy importante para nosotros en Guanaco Devs y la respetamos de una manera clara y transparente. El propósito de este documento es exponer y explicar la necesidad de nuestras aplicaciones de recolectar cierta información cuando usa las aplicaciones.

TIPO DE DATOS COLECTADOS

Nuestras aplicaciones no reúnen datos de manera predeterminada. Nuestras aplicaciones pueden recolectar datos en las siguientes situaciones:

a) INDIRECTAMENTE

Nuestras aplicaciones se basan en los servicios brindados por terceros como Google, Firebase y Facebook, por lo que ellos podriaN recopilar información sobre el identidad de su dispositivo para brindarle una experiencia más personalizada en el caso de publicidad, análisis de datos y/o almacenamiento de datos.

Firebase proporciona a Guanaco Devs almacenamiento de base de datos, procesamiento de datos, análisis y autenticación de usuario (cuando lo solicite el usuario). Los datos administrados por nuestras aplicaciones en los Servicios de Firebase se guardan bajo un Identificador de usuario aleatorio no personal. Firebase es, desde Octubre del 2014, una filial de Google.

Google proporciona a Guanaco Devs con la infraestructura necesaria para publicar y mantener nuestros productos en Google Play. Cuando el usuario realiza transacciones financieras originadas por nuestros productos, todos los datos personales, como detalles de la tarjeta de crédito, Dirección física y/o ubicación se mantienen dentro de la Infraestructura de servicios de Google, Google también proporciona un proceso de inicio de sesión fluido para personalizar el producto para el usuario.

Facebook ofrece a Guanaco Devs un proceso de inicio de sesión fluido para personalizar el producto para el usuario, así como análisis de datos dentro de su infraestructura y alojamiento para obtener información del producto dentro de sus "Páginas de Facebook": Drops Of Knowledge, Ultimate Bingo Verifier.

b) DIRECTAMENTE

Nuestras aplicaciones utilizan Firebase para ofrecer autenticación para dar a el usuario una experiencia mas personalizada usando ya sea Google o Facebook como Proveedores de Identidad Federados. Cuando el usuario opta por usar uno de los Proveedores de Identidad, recibirá y procesará

i) Dirección de correo electrónico: La dirección de correo es el principal identificador dentro de cualquiera de los proveedores de identidad federados, desde junio 15, 2018, utilizamos la dirección de correo con el único motivo de identificación validada para Gotas de Conocimiento y para, Ultra Bingo Verificador la dirección de correo es usada para enlazar las compras dentro de la aplicación con la cuenta del usuario que originó la compra.
A menos que sea explícitamente requerido no utilizamos la dirección con motivos de comunicación a menos que el usuario pida asistencia y/o información de nuestra parte.

ii) Nombre público: El nombre público es usado por la aplicación para hacer la aplicación más personalizada y para diferenciar su cuenta de otros usuarios utilizando el mismo dispositivo y desde Junio 12 del 2018, este dato es solamente utilizado por Gotas de Conocimiento. Tenga en cuenta que el nombre público no es necesariamente su nombre real.

iii) La dirección URL de la imagen de perfil: Nuestras aplicaciones pueden usar la dirección URL de la imagen de perfil para mostrar su imagen de perfil dentro de la aplicación y desde Junio 12 del 2018, este dato solamente es usado por Gotas de Conocimiento.

Todos los datos antes mencionados son utilizados dentro de la aplicación y en ningún otro lugar, estos datos son parte integral de su identidad dentro de los Proveedores de Identidad Federados(Google, Facebook).
Si usted se localiza dentro del Área Económica Europea, estamos obligados por ley a pedir de antemano sus preferencias de publicidad antes de el uso de nuestras aplicaciones.

SEGURIDAD Y MANEJO DE DATOS

Nosotros como Guanaco Devs estamos comprometidos a salvaguardar toda y cualquier información personal que usted nos confíe, puede confiar que no compartimos ni compartiremos con terceros sus datos personales a menos que sea requerido por ley, en tal caso usted será inmediatamente informado. De cualquier manera note que el mejor guardián de la seguridad de sus datos personales es usted mismo. Le pedimos encarecidamente que visite los enlaces que compartimos en la siguiente sección de este documento, para tener un mejor entendimiento de como mantener sus datos personales a salvo y cómo controlar cómo sus datos son utilizados por nuestros socios.

RETENCIÓN DE DATOS

A partir del 12 de junio del 2018 la única aplicación que guarda datos personales en nuestra Base de Datos Firebase es Gotas de Conocimiento, si por alguna razón usted decida borrar sus datos de nuestra base de datos, hemos colocado controles dentro de la configuración de la aplicación que le permite borrar TODOS sus datos personales y registros producidos y asociados con su cuenta y dispositivo(s). Por favor note que esta acción no se puede deshacer, una vez que decide eliminar sus datos de la aplicación estos no podrán ser recuperados. Si usted decide volver a usar la aplicación su progreso será inicializado.
En el evento que usted deje de usar continuamente la aplicación, los datos recolectados por el uso de la aplicación serán eliminados después de 14 meses desde la última vez que ingreso a la aplicación.
